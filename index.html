The Bootstrap GRID System


We first start by adding the following tag inside each HTML page

  1. Bootstrap 4 support for mobile-first is supported by Flexbox style
  2. Each row in a grid is divided into 12 columns
    • .col-*
    • .col-sm-*
    • .col-md-*
    • .col-lg-*

I.E: col-12 col-sm-6 means that for extra-small VPs the content would be 12 columns while starting from small up to XL viewport the content will be 6 columns

I.E: col-12 col-sm-4 col-md-3 means that for extra-small VPs the content would be 12 columns, while from small up to medium would be 4 and from medium up to extra large VPs would be 3 columns

If inside a RAW div we have the following:
col-sm-4 col-md-3
col col-sm col-md
col col-sm col-md means that we will occupy the remaining space left

col-auto will automatically manage the space or ramining portion in the GRID that must be filled

order-sm-last & order-sm-last can be applied to any div to position elements

offset-1 will use an eventual column left empty automatically

Working with NPM Scripts

npm run could be useful to run our own scripts

In this example we will:

  1. Watch for changes to the styles.scss file and automatically compile it to the css file
  2. Run multiple NPM scripts in parallel using parallelshell NPM module.

Let's start

  1. Frst, we install two NPM packages onchange and parallelshell as follows:
    1. npm install onchange --save-dev
    2. npm install --save-dev parallelshell@3.0.1 VERSION 3.0.1
  2. Add the following to our package.json scripts sections:
    1. "watch:scss": "onchange '6-Boostrap-Training/*.scss' -- npm run scss"
    2. "watch:all": "parallelshell 'npm run watch:scss' 'npm run start'"

Cleaning up a Distribution Folder

  1. Install the rimraf npm module by typing the following at the prompt:
    1. npm install rimraf --save-dev
  2. Then, set up the following script:
    1. "clean": "rimraf dist"


Copying Fonts

  1. Your project uses font-awesome fonts. These need to be copied to the distribution folder. To help us do this, install the copyfiles NPM module globally as follows:
    1. sudo npm -g install copyfiles
  2. Then, set up the following script:
    1. "copyfonts": "copyfiles -f node_modules/font-awesome/fonts/* dist/fonts"


Compressing and Minifying Images

  1. We use the imagemin-cli NPM module to help us to compress our images to reduce the size of the images being used in our project. Install the imagemin-cli module as follows:
    1. sudo npm -g install imagemin-cli
  2. Then, set up the following script:
    1. "imagemin": "imagemin 6-Boostrap-Training/img/* --out-dir='dist/img'"
  3. NOTE students have encountered issues with imagemin-cli not installing its plugins due to issues with global permissions on Mac. In that case try:
    1. sudo npm install -g imagemin-cli@3.0.0 --unsafe-perm=true --allow-root


Preparing the Distribution Folde
