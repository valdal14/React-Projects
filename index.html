The Bootstrap GRID System


We first start by adding the following tag inside each HTML page

  1. Bootstrap 4 support for mobile-first is supported by Flexbox style
  2. Each row in a grid is divided into 12 columns
    • .col-*
    • .col-sm-*
    • .col-md-*
    • .col-lg-*

I.E: col-12 col-sm-6 means that for extra-small VPs the content would be 12 columns while starting from small up to XL viewport the content will be 6 columns

I.E: col-12 col-sm-4 col-md-3 means that for extra-small VPs the content would be 12 columns, while from small up to medium would be 4 and from medium up to extra large VPs would be 3 columns

If inside a RAW div we have the following:
col-sm-4 col-md-3
col col-sm col-md
col col-sm col-md means that we will occupy the remaining space left

col-auto will automatically manage the space or ramining portion in the GRID that must be filled

order-sm-last & order-sm-last can be applied to any div to position elements

offset-1 will use an eventual column left empty automatically
