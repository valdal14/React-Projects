Git Commands

Useful Git Commands


// Initilize a folder as Master branch

git init

// shows the heads and commit comments about this repo

git log --oneline

I.E:ae2b01f (HEAD -> main, origin/main, origin/HEAD) Added the stylings to the home page
f1e6922 Based starting project
7086dfe Initial commit

We can see we have done three commits so far. Now let's see how we can rollback to a previous commit

// checkout a file from a previous commit

git checkout f1e6922 index.html

// reset a checkout version inside the staging area

git reset HEAD index.html

// sync a local repo with a remote repo

git remote add origin https://github.com/valdal14/React-Projects.git

// push a local repo files to the a remote repo

git push -u origin master
